### 2. 적성검사 질문 3세트 구조 - `src/data/aptitudeQuestions.ts`

- 새 타입: `QuestionSetId = 'set1' | 'set2' | 'set3'`, `AptitudeQuestionSet`
- 새 헬퍼: `getQuestionSet(previousSetId?)` - 이전과 다른 세트 자동 선택
- `calculateResult(answers, questions)` - questions 파라미터 추가
- 하위호환 유지: `export const QUESTIONS = QUESTION_SETS[0].questions`
- 각 세트 15문항, CEO/PM/CPO/CMO/CSL 균등 가중치 분배
- i18n 키: `school.aptitude.questions.set1.Q1` ~ `set3.Q15`

### 3. AI 프롬프트 + mock 데이터 - `src/services/gemini/marketCompassService.ts`
- `generateMarketAnalysis()`: 4번째 파라미터 `itemType` 추가 (default: 'other')
  - 프롬프트에 아이템 형태 포함, `analysisReport` 필드 요청 (20줄, 쉬운 한국어)
- `generateBrandingStrategy()`: 3번째 파라미터 `competitors: CompetitorInfo[]` 추가
  - 프롬프트에 경쟁사 정보 포함, `brandingReport` 필드 요청 (20줄, 쉬운 한국어)
- mock 데이터 3종 (food, beauty, default) 모두 20줄 `analysisReport` 포함
- mock 브랜딩 데이터도 20줄 `brandingReport` 포함

### 4. 적성검사 도구 페이지 - `src/pages/marketing/school/tools/AptitudeTestTool.tsx`

### 5. 마켓 스캐너 도구 페이지 - `src/pages/marketing/school/tools/MarketScannerTool.tsx`

  - `whitespace-pre-line` + 복사 버튼
