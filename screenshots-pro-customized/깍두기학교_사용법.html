깍두기학교 사용법

AI 마케팅 도구 학교 6교시 + 프로 모드 6개
비건 쿠키 예시로 처음부터 끝까지 보여드려요

학교 도구

마케팅학과 6교시

1교시부터 6교시까지, 마케팅의 기초를 AI와 함께 배워요.
각 도구에 상품 정보를 넣으면 AI가 자동으로 분석하고 만들어줘요.

📝 1교시 — 적성검사

나에게 맞는 마케팅 스타일을 찾아줘요. 간단한 질문에 답하면 AI가 내 마케팅 성향을 분석해요.

사용 방법

  1. 마케팅학과에서 1교시 "적성검사" 클릭
  2. 화면에 나오는 질문에 답하기 (5~6개)
  3. 결과 확인! — 내 마케팅 유형이 나와요

질문에 답하는 화면

내 마케팅 유형 결과

🔍 2교시 — 시장탐색기

내 상품의 시장을 AI가 분석해요. 경쟁사, 고객의 고민, 관련 키워드를 한눈에 볼 수 있어요.

사용 방법

  1. "내 상품" 칸에 팔고 싶은 물건 입력 (예: 비건 쿠키)
  2. 고객 나이, 성별, 상품 종류 입력
  3. "시장 분석 시작" 버튼 클릭
  4. AI가 경쟁사, 고객 고민, 키워드를 알려줘요

상품 정보 입력

AI 분석 결과

⚡ 3교시 — 브랜드만들기

내 브랜드의 강점(USP)과 슬로건을 AI가 만들어줘요. 경쟁사와 다른 "우리만의 매력"을 찾아요.

사용 방법

  1. 내 상품의 장점을 입력
  2. "브랜드 만들기" 버튼 클릭
  3. AI가 USP(우리만의 매력) + 슬로건을 만들어줘요
  4. 마음에 들면 보석함에 저장!

장점 입력

USP + 슬로건 결과

📱 4교시 — 콘텐츠메이커

SNS에 올릴 광고 카드를 AI가 만들어줘요. 문구와 디자인이 자동으로 나와요.

사용 방법

  1. 상품 이름과 고객 정보 입력
  2. "광고 카드 만들기" 버튼 클릭
  3. AI가 카드 여러 장을 만들어줘요
  4. 마음에 드는 카드를 PNG로 저장

상품 정보 입력

AI 광고 카드

📋 5교시 — 세일즈라이터 (상세페이지 + 라이브 대본)

쇼핑몰 상세페이지와 라이브 방송 대본을 AI가 만들어줘요.

사용 방법

  1. 상품 이름, 고객, 장점 입력
  2. "상세페이지" 또는 "라이브 대본" 선택
  3. AI가 실제 사용 가능한 내용을 만들어줘요
  4. 텍스트를 클릭하면 직접 수정도 가능!

입력 화면

상세페이지 결과

📊 6교시 — ROAS 예측기

광고비 대비 매출(ROAS)을 예측해요. 광고에 돈을 얼마 쓰면 얼마 벌 수 있는지 AI가 계산해요.

사용 방법

  1. 광고비, 상품 가격 입력
  2. "예측하기" 버튼 클릭
  3. AI가 예상 매출과 수익을 계산해줘요
  4. 차트로 한눈에 비교!

광고비 입력

수익 예측 결과

🎓 수료증

6교시를 모두 마치면 수료증을 받을 수 있어요!

PRO

프로 모드

학교를 졸업하면 열리는 고급 도구예요.
더 자세한 분석, 커스텀(내 맘대로 수정), PDF 저장이 가능해요.

📊 시장 리서치 프로

시장 크기, 경쟁사, 강점/약점, 기회, 고객, 진입 전략까지 6가지를 AI가 깊이 분석해요. 결과를 PDF로 저장할 수 있어요.

사용 방법

  1. 상품/업종, 고객 정보 입력
  2. "시장 분석 리포트 만들기" 클릭
  3. 6가지 요약 카드가 먼저 나와요
  4. "자세한 레포트 보기" 버튼을 누르면 전체 리포트가 펼쳐져요
  5. 각 섹션을 수정할 수 있고, PDF로 저장 가능!

정보 입력

요약 카드 + "자세한 레포트 보기" 버튼

자세한 레포트 (수정 가능 + PDF 다운로드)

🎨 브랜드 키트 프로

브랜드만의 매력 문장 + 슬로건 + 색상 팔레트 + 글꼴을 한번에 만들어요. 색상과 글꼴을 직접 바꿔볼 수 있어요!

사용 방법

  1. 브랜드 이름, 업종, 고객, 느낌 입력
  2. "브랜드 키트 생성" 클릭
  3. "자세한 레포트 보기"를 누르면 전체가 열려요
  4. 색상을 클릭해서 바꿔보세요 (컬러 피커)
  5. 글꼴도 바꿔볼 수 있어요 (제목/본문 따로)
  6. 슬로건도 직접 수정 가능!

요약 + 컬러 미리보기

색상·글꼴·슬로건 커스텀 적용

📱 콘텐츠 스튜디오 프로

AI가 광고 카드를 만들어주고, 글자 크기·색상·배경을 직접 바꿀 수 있어요. 사진도 넣을 수 있어요!

사용 방법

  1. 상품 이름, 고객, 톤(감성/재미/정보) 선택
  2. 카드 수와 사이즈 선택 후 "광고 카드 생성" 클릭
  3. 각 카드마다 수정 가능:
  4. — 글자 크기 슬라이더로 조절
  5. — 글자색, 배경색 변경
  6. — 사진 업로드 + 오버레이 투명도 조절
  7. 전체 PNG로 다운로드!

톤 선택 + 입력

글자 크기·색상 커스텀 적용

📦 상세페이지 프로

쿠팡/스마트스토어 스타일 상세페이지를 AI가 만들어요. 사진을 넣고 텍스트를 수정해서 바로 쓸 수 있어요!

사용 방법

  1. 상품명, 고객, 장점, 할인 혜택 입력
  2. "상세페이지 생성" 클릭
  3. 모바일 프리뷰가 나와요 — 텍스트를 클릭하면 직접 수정!
  4. "이미지 추가" 버튼으로 상품 사진 넣기
  5. 완성되면 PDF로 저장

정보 입력

AI 결과 + 이미지 삽입

🛒 랜딩페이지 빌더 프로

실제 홈페이지처럼 생긴 랜딩페이지를 만들 수 있어요. 섹션을 추가/삭제하고 순서도 바꿀 수 있어요.

사용 방법

  1. 상품명, 고객, 장점, 혜택 입력
  2. "랜딩페이지 생성" 클릭
  3. 섹션별로 내용 수정, 배경색 변경, 이미지 업로드
  4. 구매 버튼(CTA) 텍스트와 색상도 바꿀 수 있어요
  5. 모바일/태블릿/데스크톱 미리보기 가능
  6. PDF로 저장!

정보 입력

배경색·CTA 버튼 커스텀

📈 마케팅 대시보드 프로

월별 광고비와 매출을 입력하면 차트로 보여주고, AI가 개선점을 알려줘요.

사용 방법

  1. 목표 ROAS(광고 대비 매출 배수) 설정
  2. 매달 광고비와 매출 입력 → "추가" 클릭
  3. 차트가 자동으로 그려져요 (매출/ROAS/채널별)
  4. "AI에게 분석 & 조언 받기" 클릭
  5. "자세한 레포트 보기"로 전체 분석 확인
  6. 월간 리포트 PDF 저장!
