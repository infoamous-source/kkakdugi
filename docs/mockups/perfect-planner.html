  3. 가격 표시 강화 — 30% 빨간 배지 + 정가 취소선 + 큰 빨간 할인가 + 카운트다운
  4. 헤드라인 톤 변경 — "3분이면 카페가 우리집?!" 처럼 캐주얼·구어체, 큰 검은 배경 흰 글씨
  5. Pain point가 "~한 사람!" 형태 — 노란 배경 + 흰 카드 + 좌측 노란 바
  6. 어그로 섹션 AI 자동 선택 (B형/C형) — 콘텐츠 성격에 따라 어그로형("✋ 스크롤 내리기 전에") 또는 사회적 증거형("이미 2,341명이 선택한 이유 ↓")을 자동 선택. 두 형태 모두 mockup에 표시함
  7. 3가지 특징 큰 블록 — 컬러 코딩(노랑/초록/파랑) + 큰 이모지 + 짧은 한 줄
  8. 리뷰 카드 + 자정 카운트다운 + 빨간 마지막 강조
  9. 학습카드/프레임워크 라벨 제거
  10. 라이브 방송 모드 전환 시 — 진짜 큐시트 형식 (시간·진행자 멘트·액션·시청자 반응) — 펼치기 항목으로 표시
