변경된 점

  1. 시뮬레이터 → 자가 진단기 — AI가 매출을 예측하는 게 아니라, 학생이 본인 광고비·매출을 직접 넣고 신호등으로 진단받는 방식
  2. 입력 칸 단 3개 — ① 광고비 ② 매출 ③ 채널. 끝. (가격·연령·예산 다 제거)
  3. 큰 ROAS 숫자 (54px) + 한 줄 의미 — "광고비 1원 → 매출 1.8원" 으로 ROAS 개념 자연 학습
  4. 신호등 (📉손해 / 😐본전 / 📈좋아요) — 3개 동그라미 중 1개만 컬러로 빛남, 나머지는 회색
  5. 큰 한 줄 처방 — "👇 광고비를 줄이세요 / 30만원 → 20만원" 으로 액션 명확
  6. 오늘 할 일 1개만 — 3개가 아니라 가장 중요한 1개
  7. 자세한 숫자는 4개로 단순화 — 광고비 / 매출 / 남은 돈(이익) / ROAS. 노출·클릭·CVR·CPC 전부 제거
  8. 계산식 한 줄 노출 — "매출 540,000 ÷ 광고비 300,000 = 1.8" 수업에서 설명 가능
  9. 다시 해보기 버튼이 신호등을 바꿈 — 학생이 광고비/매출 바꿔가며 신호등 변화 직접 체험
  10. AI 의존도 감소 — ROAS는 수식 계산. AI는 처방·오늘 할 일만 가볍게 호출 → 30명 동시 부하 안전
