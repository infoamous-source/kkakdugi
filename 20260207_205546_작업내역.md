## 해결된 이슈

### Bash 템플릿 문법 에러
- **문제**: `MarketingModuleCard.tsx`를 heredoc(<<)로 생성 시 `${gradientMap[stageColor]}` 같은 TypeScript 템플릿 리터럴이 Bash 변수 치환으로 해석됨
- **원인**: Bash heredoc 내에서 `${}` 구문이 shell substitution으로 처리
- **상태**: 파일 생성 실패 → 다음 세션에서 Write 도구로 재생성 필요
